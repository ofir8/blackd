# Black Diamond - סוויטה אקסלוסיבית בגליל

אתר יוקרתי לאירוח בכפר ורדים, גליל מערבי.

## טכנולוגיות

הפרויקט נבנה עם:

- **Vite** - Build tool מהיר
- **React 18** - ספריית UI
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **shadcn/ui** - רכיבי UI
- **React Router** - ניווט
- **Framer Motion** - אנימציות

## התקנה והרצה

```bash
# התקן dependencies
npm install

# הרץ שרת פיתוח
npm run dev

# בנייה ל-production
npm run build

# תצוגה מקדימה של build
npm run preview
```

## מבנה הפרויקט

```
src/
├── components/     # קומפוננטות React
├── pages/          # דפי האתר
├── config/         # קבצי הגדרות
├── assets/         # תמונות וקבצים סטטיים
└── hooks/          # Custom hooks
```

## פריסה ל-GitHub Pages

האתר מוגדר לפריסה אוטומטית ל-GitHub Pages באמצעות GitHub Actions.

ראה `GITHUB_PAGES_SETUP.md` להוראות מפורטות.
